Recipe Guidelines
Creating a Recipe
  1. Enter a descriptive recipe name (e.g., "Croissant")
  2. Select the final product this recipe produces
  3. Specify how much the recipe yields
  4. Add a description (optional)
  5. Add all required ingredients with quantities
  6. Write the preparation steps
Example

Recipe Name: Croissant

Ingredients:

  • Flour - 500g
  • Butter - 250g
  • Sugar - 50g
  • Salt - 10g

Steps: Mix, knead, fold, bake...

Tips
  • Use clear, descriptive names for recipes
  • Ensure yield quantities are accurate
  • Add all ingredients before saving
  • Write detailed preparation steps
